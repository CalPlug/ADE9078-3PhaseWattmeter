How to use CAM file for Board Gerber files
1. Look over for board, check with DRC features
2. File -> CAM Processor
3. Do not use the template.cam on the top right
4. Open from the CAM Processor the file: 
	"Smart_Prototyping_NEW_2_layer_with_name.cam"
5. Check Gerber layers and check if all is correct
	*Make sure Vcut has Layer 45 Holes and Layer 46 Milling
6. Check the box "Export as ZIP"
7. Click on Process Job on the bottom right
